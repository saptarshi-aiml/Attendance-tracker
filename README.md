# 📊 Attendance Tracker

A simple and user-friendly web application to calculate attendance percentage and track academic eligibility.

---

## 🚀 Features

- Calculate attendance percentage
- Instant status check:
  - ✅ Safe (≥ 75%)
  - ⚠️ Warning (60–74%)
  - ❌ Low (< 60%)
- Input validation
- Clean and responsive UI

---

## 🧠 Formula

Attendance % = (Attended Classes / Total Classes) × 100

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## ▶️ How to Use

1. Enter total classes
2. Enter attended classes
3. Click "Calculate"
4. View percentage and status
